# Attendance Calculator

A simple **Attendance Calculator** built using **HTML, CSS, and JavaScript** to calculate attendance percentage instantly.

## Features
- Enter days present and days absent
- Calculates attendance percentage
- Shows attendance status (Excellent / Good / Average / Poor)
- Progress bar visualization
- Reset option

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open `index.html` in a web browser.
2. Enter the number of days present and absent.
3. Click **Calculate** to view attendance details.
4. Click **Reset** to clear inputs.
